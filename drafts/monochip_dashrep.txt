heading-begin
The Revolution that will End Computer Central Processing Units
heading-end

paragraph-begin
Smartphones, and computers, waste huge amounts of time and electricity.
sentence-next
Why?
sentence-next
All the calculations are done in central processing units.
sentence-next
As the word italics-begin central italics-end suggests,
all the numbers being processed have to wait their turn
getting to and from the central processing unit.
sentence-next
Yes there are lots of ways to speed up that transfer of data.
sentence-next
Yet that bottleneck is still there.
paragraph-end

paragraph-begin
The secret to making mobile devices that run much, much faster,
with less waste of battery power,
is to distribute simple processing units throughout
each and every computer chip.
sentence-next
This means each small group of memory locations will have their own
simple processing unit.
paragraph-end

paragraph-begin
It's like ...

paragraph-end

subheading-begin
Serial processing units, not central processing units
subheading-end



subheading-begin
Each Memory Word Gets Its Own Serial Processing Unit
subheading-end




subheading-begin
Chip defect tolerance
subheading-end

paragraph-begin
The chip's huge number of simple processing units allows lots of them to have chip defects.
sentence-next
The runtime compiler can avoid using the units that have a defect.
sentence-next
This tolerance will dramaticlly reduce the manufacturing cost.
paragraph-end


subheading-begin
Programming a serial processing unit
subheading-end

paragraph-begin
Instead of instructions, a string of bits is sent to the input side of the serial processing unit, and those bits select which inputs are supplied to that serial processing unit.
paragraph-end

paragraph-begin
Each serial processing unit is a one-bit full adder. sentence-next It has an A and B input and a carry input. sentence-next It has a main one-bit output and a carry output. sentence-next 

paragraph-begin
Based on what is supplied to the A, B, and carry inputs, the main one-bit output of the full adder can calculate these one-bit results:
paragraph-end

list-begin
ADD
list-next
OR
list-next
XOR
list-next
NOT
list-end

Using the above calculations can yield the following numbers:
paragraph-end

As explained above, a string of bits sent to the input side of the serial processing unit select which inputs are supplied to that serial processing unit. sentence-next Here are all the possible inputs. sentence-next Only some of them are available to only some of the single-bit full-adder units.
paragraph-end

list-begin
from_adder_above
list-next
from_adder_left
list-next
from_adder_right
list-next
from_adder_left_pairwise_N sentence-next where N can be 0 through 31
list-next
from_zero
list-next
from_one
list-next
from_one_initial sentence-next which increments the first bit within a number, and afterward is zero
list-end

paragraph-begin
The 32 pairwise inputs are used for multiplication using the method described below. sentence-next This input option is available in only some columns (if data flow is top to bottom) or only some rows (if data flow is left to right).
paragraph-end

paragraph-begin
With this simple data processing it is possible to do integer addition, integer subtraction, integer multiplication, and integer division.
paragraph-end

paragraph-begin
When decimal numbers are needed, either in the compiler or at runtime, the decimal numbers can be scaled and offset to fit between zero and the largest integer, then after calculating using those integer numbers, the integers can be converted back into decimal numbers.
paragraph-end

paragraph-begin
When floating-point numbers are used, either the compiler or the runtime code can manage the sign (plus or minus) and the exponential portion. sentence-next Addition, subtraction, multiplication, and division are done using just the integer portion. sentence-next Then the three parts (sign, mantissa, and exponent) are combined into the standard single floating-point representation.
paragraph-end

All data transfers copy the data without erasing the content of the source location, and without saving the previous content of the destination location.

paragraph-end
If power consumption should be minimized, the chip type is capacitor-based "latches" similar to dynamic memory chips.
sentence-next
This dramatically reduces transistor count because a latch requires many transistors.
sentence-next
In contrast, there are just two transistors between each capacitor.
sentence-next
Those transistors "amplify" the capacitor state at the input and charge or discharge the capacitor at the output.
sentence-next
(This might slightly complicate the compiler because it becomes responsible for transferring a capacitor's state before it loses its charge, but that's easy to handle because not all the groups of 16 registers are always being used for storage and processing.)
paragraph-begin
